## Gets the number of active users on a website.
**GET /api/websites/:websiteId/active**

**Parameters** \
None


---

**Sample route**
```
/api/websites/xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx/active
```

**Sample response**
```json
{
  "x": 5
}
```
x: Number of unique visitors within the last 5 minutes
